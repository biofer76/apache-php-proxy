

Apache Web Server and PHP Engine with Proxy modules
--------------------------------

    AJP (proxy\_ajp)   
    Balancer  (proxy\_balancer)    
    Connect (proxy\_connect)   
    FastCGI (proxy\_fcgi)  
    FTP (proxy\_ftp)  
    HTML (proxy\_html)  
    HTTP (proxy\_http)  
    xml2enc (recommend module)
